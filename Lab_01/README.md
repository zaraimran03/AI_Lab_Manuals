**🎯 Objective:**  
Learn how to create and upload datasets in Python, perform basic statistical analysis, and visualize data using graphs.

**📚 Topics Covered:**
- Manual dataset creation with Pandas
- CSV file operations
- Statistical analysis (mean, max, correlation)
- Data visualization techniques

**🔧 Key Technologies:**
- `pandas` - Data manipulation
- `matplotlib` - Visualization

4. ✔ Calculated statistics (mean Math marks, max Science marks)
5. ✔ Filtered students with Math marks > 50
6. ✔ Found student with highest Science marks
7. ✔ Calculated correlation between Math and Science marks

   - Bar chart: Student ID vs Math Marks

   - Scatter plot: Math vs Science marks
